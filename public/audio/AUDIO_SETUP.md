# 🎵 Audio Setup — Raindrops (An Angel Cried)

Place your audio file here:

```
/public/audio/raindrops (an angel cried).mp3
```

This file plays **once** when the gates finish opening — crystal clear, no loop, no repeat.

If the file is missing, the site loads silently without errors (the ambient toggle still works independently).

## Ambient Background Audio
The ambient loop uses:
```
/public/audio/album_dj_em/Electric Whisper.mp3
```
Users can toggle it on/off with the bottom-right button.
